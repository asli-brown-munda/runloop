# Tasks

This is the running task list for Runloop, split by area. Each file in this directory holds the open tasks for one area, with a checklist of named tasks and short descriptions.

The current baseline is captured in `docs/current_state.md`. The intentional non-goals (DAGs, distributed execution, remote control plane, etc.) are listed in `docs/architecture.md` and `docs/mvp.md` and are explicitly out of scope here.
